# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-12-17

### Adicionado
- Assistente Financeiro Inteligente com IA Generativa
- Sistema de FAQ inteligente com busca semântica
- Calculadoras financeiras:
  - Juros compostos
  - Financiamento (Price e SAC)
  - Investimentos (CDB, Poupança, Tesouro Direto)
  - ROI (Retorno sobre Investimento)
- Sistema de análise de dados financeiros:
  - Upload e processamento de CSV
  - Estatísticas descritivas
  - Detecção de outliers
  - Análise de tendências
- Chatbot conversacional com contexto persistente
- Gerenciamento de sessões de usuário
- Interface Streamlit interativa
- Documentação completa:
  - README detalhado
  - Guia de contribuição
  - Política de segurança
  - Código de conduta
  - Arquitetura do sistema
  - Documentação da API
  - Guia de deployment
  - Princípios de UX
- Testes unitários abrangentes
- Exemplos de dados para demonstração
- Scripts auxiliares de inicialização

### Segurança
- Implementação de gerenciamento seguro de chaves API
- Validação de entrada de dados
- Proteção contra injeção de código
- Sanitização de uploads de arquivo
